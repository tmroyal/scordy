# scordy

Click [tmroyal.com/scordy](https://tmroyal.com/scordy) to see a live demo.

Repo for scordy, a block code creative computer sequencer/visualizer.

This was created using Blockly, p5.js, the WebAudio API, and coffee.

If this project is interesting to you, please contact me with any feedback so that I may know whether to further refine this project.
